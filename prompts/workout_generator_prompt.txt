<zwoFormatRequirements>
ZWO files are XML-based workout files for Zwift with the following structure:

Root element: <workout_file>
Required metadata elements:
  - <name>Workout Name</name>
  - <author>Train-R</author>
  - <description>Brief workout description</description>
  - <sportType>bike</sportType>

Workout container: <workout>
Inside <workout>, use these elements to build the workout:

1. Warmup - Gradual power ramp
   <Warmup Duration="SECONDS" PowerLow="0.4" PowerHigh="0.75"/>
   Duration: time in seconds
   PowerLow/PowerHigh: power as decimal % of FTP (0.4 = 40% FTP)

2. Cooldown - Gradual power decrease
   <Cooldown Duration="SECONDS" PowerLow="0.75" PowerHigh="0.3"/>
   Duration: time in seconds
   PowerLow/PowerHigh: start and end power as decimal % of FTP

3. SteadyState - Constant power interval
   <SteadyState Duration="SECONDS" Power="0.88"/>
   Duration: time in seconds
   Power: target power as decimal % of FTP

4. Ramp - Power ramp between two values
   <Ramp Duration="SECONDS" PowerLow="0.5" PowerHigh="1.05"/>
   Duration: time in seconds
   PowerLow/PowerHigh: power range as decimal % of FTP

5. IntervalsT - Repeated on/off intervals
   <IntervalsT Repeat="5" OnDuration="180" OffDuration="120" OnPower="1.2" OffPower="0.5"/>
   Repeat: number of repetitions
   OnDuration/OffDuration: work and rest duration in seconds
   OnPower/OffPower: work and rest power as decimal % of FTP

6. FreeRide - No specific power target
   <FreeRide Duration="SECONDS"/>
   Duration: time in seconds

Power guidelines (as decimal % of FTP):
- Active Recovery: 0.3-0.55
- Endurance: 0.55-0.75
- Tempo: 0.75-0.87
- Sweet Spot: 0.88-0.94
- Threshold: 0.95-1.05
- VO2 Max: 1.06-1.20
- Anaerobic: 1.21-1.50
- Sprint: 1.50+
</zwoFormatRequirements>
